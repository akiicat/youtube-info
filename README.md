# Youtube Retriever

## Installation

```yml
dependencies:
  youtube_retriever:
    github: akiicat/youtube_retriever
    branch: master
```

## Usage

```cr
require "youtube_retriever"

YoutubeRetriever.dump_json("iDfZua4IS4A")
```
